# ☁️ Cloud Web Server Deployment on AWS EC2

## Project Overview
This project demonstrates how to deploy a web server on AWS EC2 and host a website in the cloud. It also focuses on configuring secure access to the server using AWS security features.

---

## Objectives
- Deploy a virtual server using AWS EC2  
- Install and configure Apache web server  
- Host a webpage on the cloud  
- Manage secure access using security groups  

---

## Technologies Used
- AWS EC2  
- Ubuntu Server  
- Apache Web Server  
- HTML  
- SSH  
- Security Groups  

---

## Architecture
User → Internet → AWS Cloud → EC2 Instance → Apache Web Server → Webpage

---

## Implementation Steps

1. Launch an EC2 instance (Ubuntu)  
2. Configure key pair and security groups  
3. Connect to instance using SSH  
4. Install Apache web server  
5. Deploy HTML webpage  
6. Access website using public IP  

---

## Output
The hosted webpage can be accessed through the EC2 public IP address in a web browser.

---

## Security
- Security groups control inbound and outbound traffic  
- SSH key ensures secure login  
- Only authorized access is allowed  

---

## Conclusion
This project demonstrates the practical implementation of cloud computing by deploying and managing a web server on AWS EC2.
